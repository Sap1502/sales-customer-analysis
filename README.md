# Sales and Customer Analysis using Python

## Project Overview
This project analyzes a retail company's sales and customer data to identify trends, customer behavior, and revenue insights.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Key Analysis
- Total sales and revenue trends
- Sales by product category
- Monthly sales analysis
- Top customers identification
- Customer demographic analysis

## Insights
- Certain product categories drive maximum revenue
- Seasonal trends affect sales
- Loyal customers contribute significantly to total sales

## Conclusion
The analysis helps businesses make data-driven decisions related to marketing, inventory, and customer targeting.
